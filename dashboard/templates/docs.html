python3 -m venv venv

❓ FAQ

Does it work on Windows?

Yes! You can run Aegis natively with Python, or use Docker for a containerized setup. WSL2 also works great.

Which AI model works best?

We recommend phi3:mini — it's fast, lightweight, and runs well on modest hardware.

Is my email password stored?

Never! Your password creates an AES-256 encryption key via PBKDF2 (600K iterations). The key is derived at runtime and never written to disk.

Does it need internet?

No. Aegis is 100% offline. All AI processing happens locally via Ollama. Emails are downloaded via IMAP but never sent anywhere else.
